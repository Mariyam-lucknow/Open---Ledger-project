# Open---Ledger-project
The Open Ledger Project is a free behavioral economics-based budgeting tool for Indian middle-class families.

Built on 5 principles: Thaler's Mental Accounting, Mullainathan's Bandwidth Tax, Kahneman's System 1 vs System 2, Thaler's Default Effect, and Festinger's Cognitive Dissonance.

How to use: Fill the form once at month end. Takes 10 minutes.

Live tool: [https://forms.gle/8NemPiGA29ECBiy68]

Known limitation: Google Forms does not support mid-session saves. This is a monthly single-session reflection tool, not a real-time expense tracker. This limitation is acknowledged in the design.

Built by Mariyam, Lucknow, India. Inspired by observing financial decision-making under scarcity in Thakurganj."
