The Open Ledger Project

A free behavioral economics-based budgeting tool for Indian middle-class families.

Built on 5 principles: Thaler's Mental Accounting, Mullainathan's Bandwidth Tax, Kahneman's System 1 vs System 2, Thaler's Default Effect, and Festinger's Cognitive Dissonance.

Access the tool here:
https://docs.google.com/spreadsheets/d/1H7Q9mI37PnDYOe4j5xxCrK5sbx3bKpPBSN_MYwVt3uA/edit?usp=sharing

How to use: Open the link → File → Make a copy → Save to your Google Drive → Fill your monthly numbers.

Built by Mariyam, Lucknow, India. Inspired by observing financial decision-making under scarcity in Thakurganj.
